Teste para Anlise de Dados senhasegura

1 Introduo

Este projeto tem como foco a anlise de dados de vendas de uma loja online para o teste de Estgio em Dados da senhasegura, com o objetivo principal de manipular os dados, realizar uma anlise exploratria e obter insights detalhados dos resultados.

1.1- Objetivos Especficos

  • Limpeza e Preparao dos Dados: Ser feita a verificao de dados ausentes, bem como a edio de alguns dados para melhor visualizao.

  • Analise exploratoria dos Dados: Ser realizada uma anlise aprofundada para compreend-los melhor e, assim, obter insights e resultados mais precisos.

  • Resultados e Analises Obtidos: Ao final, com todos os dados em mos, ser realizada uma anlise detalhada, seguida de algumas sugestes de melhorias.

1.2- Ferramentas e Metodologia

  • Ferramentas Utilizadas: Python e Jupyter Notebooks para anlise de dados, com bibliotecas como Pandas, Plotly, Matplotlib e Seaborn para manipulao de dados e visualizaes.
  • Abordagem de Anlise: A anlise ser conduzida por meio de uma combinao de estatsticas descritivas, visualizaes de dados e explorao de variveis especficas, que so potencialmente relevantes para o teste.

Com esses objetivos e ferramentas em mente, procederemos com a anlise detalhada dos dados fornecidos.

No caso de colunas categricas, o valor mais frequente poderia ser uma boa substituio.

Essas abordagens garantiriam que o dataset continuasse consistente e sem vieses, evitando a excluso excessiva de dados relevantes ou a introduo de valores incorretos.

2.4 - Conversao da coluna "Data da Venda" para o formato apropriado

In[5]:
# Inicializar a lista de vendas
vendas = []

# Abrir e ler o arquivo CSV
with open('dados_de_vendas.csv', mode='r', encoding='utf-8') as file:
    reader = csv.DictReader(file)
    for row in reader:
        # Converter 'Data da Venda' de string para objeto de data
        row['Data da Venda'] = datetime.strptime(row['Data da Venda'], '%Y-%m-%d')

        # Adicionar a venda  lista
        vendas.append(row)

A coluna "Data da Venda" foi inicialmente importada como string, o que impede uma anlise temporal precisa. Para corrigir isso, converti essa coluna para o formato datetime, utilizando a funo datetime.strptime(). A converso permitiu realizar anlises temporais, como a evoluo das vendas ao longo dos meses, facilitando o agrupamento por perodos (ano, ms, etc.) e a identificao de tendncias sazonais.

Essa converso foi fundamental para realizar a anlise correta de vendas mensais e sazonais, possibilitando o clculo preciso de mtricas como "valor total por ms" e "picos de vendas", insights cruciais para qualquer anlise temporal.
